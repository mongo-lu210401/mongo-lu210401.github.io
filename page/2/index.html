关于包法利夫人

发表于 2021-04-14 分类于 读书笔记

这本书可是引发我太多思考，太多疑问了。就感觉很多说不上来的东西堵在胸口，即使去豆瓣上看了一些书评也还是不能形成一个完整的叙事。书中我在包法利夫人身上看到了自己，在罗多尔夫和莱昂身上看见了他们，关于爱玛悲剧的发生我又如何去理解，接下来会是一篇毫无逻辑可言的读书笔记。

包法利夫人是否是在追求自我

拿到书之前看到简介说，这本书讲述了包法利夫人在欲望中堕落最终迷失自我的过程。可是在阅读的过程中，她对丈夫包法利先生平凡且平庸的难以忍受，对爱情始终如一的近乎“偏执”的追求，我又很真实的感受到她对自我的追求。知道自己没有被满足的，知道自己追求的（即完美的爱情），并为之奋不顾身的，难道不能算是一种对自我的追求吗？没有自我意识的人是顺从的，顺从于命运，顺从于眼前这个并不能满足自己需求的丈夫，压抑或麻痹自己的情感与追求，最终过上起码表面看起来平静的生活。可包法利夫人不是这样的呀，她清楚的知道眼前这个人无法满足自己的需求，无论是能力还是情感上，她想要得到一份完美的爱情，对方是个能让自己崇拜的人，同时两个人在一起又都全心全意的付出，产生情感上的连接甚至融合，即使找错了人，也依然没有放弃寻找，这为什么会是一种自我的迷失呢，我觉得这是一种自我的追求啊。把希望寄托于爱情有错吗？我们每个人对爱情不都有希望吗？虽然对爱情的具体希望不同，比如有的只是搭伙过日子，有的则是灵魂上的契合，但希望爱情是完美的有错吗，追求爱情是完美的不可以吗？

但另一方面，希望一个男人的出现来给予她完美的爱情，这么说又好像是有哪里不对劲了。在对丈夫平凡又平庸的一次次失望后，她幻想着一个人的出现能带给她完美的爱情，能解救她脱离眼前无聊的生活。换句话说，她希望会有一个人英雄般的出现来拯救她，而不是自己去主动的改变现状（走出婚姻关系也好，去想办法满足自己的需求也好），去发挥主观能动性的创造更多可能性，于是但凡出现一个人，她就以为自己得救了，进而倾注所有，奋不顾身投入到“完美爱情”的假象中，即使后来发现这些关系也都是【平凡且平庸】的，即使被抛弃，她依然希望着下一个拯救者的出现。这个角度来看，包法利夫人好像又是没有自我的。

理想与现实该如何去平衡

包法利夫人对完美爱情的追求应该起源于她在修道院看的爱情小说，小说里构建出的浪漫爱情成为了她后来一直想要却没得到的。虽然我们都知道电影小说里的情节都是虚构的，是不现实的，但那一定是不存在的吗？虽然大部分人称是没有见到过和经历过的，但是大部分人甚至绝大部分人没经历过就是没有吗？谁来告诉你这个东西它究竟是否真实存在呢？谁又有资格来告诉你呢？那么当我们发现现实世界和理想世界不同的时候，我们如何去看待理想国的存在，我们选择放弃理想还是放弃现实，如果选择都不放弃，又怎么去让理想落地，怎么去改变现实，让二者接轨。我想，理想与现实的平衡问题的第一步一定是清楚的认识到理想和现实的差距，即认识到什么是理想，什么是现实，至于后面的选择与平衡，可能单纯的是个人选择的问题了，或者我也不知道了。

认识一段关系认识一个人

  • 与包法利先生
    包法利夫人看不上或者可以说完全无法忍受丈夫的平庸与无能。可是一开始的时候不是这样的，开始的时候包法利夫人对包法利先生也是有感情的并且愿意嫁给她的。这种转变产生的原因我想主要还是爱玛对爱情的渴望让她在缺乏对包法利先生有比较充分了解的时候便急于嫁给他。包法利先生也从来没有刻意隐瞒自己的平庸，只是对爱情的渴望让爱玛忽视了这一点，就好像在热恋期的人们都会觉得对方特别完美一样，爱情需要时间的考验，尤其是当蜜月期过去，当一个人更加完整的暴露在你面前的时候，你才可能更清醒的认识到彼此之间的感情，而在这之前可以继续关系但不要急于推进关系。

    写到这里我突然想到我ex在跟我最后一次的谈话中说到我们关系破裂的根本在于在一起的过于唐突了，当我完整的暴露在他面前的时候他发现他承受不了。我觉得把“在一起的过于唐突了”作为我们关系的总结实在是过于表面而且草率甚至有点不负责了。也许我们在一起的时候是还没有完全的了解彼此，但所有的感情都是在等到完全的了解了彼此才开始的吗？那又是后来慢慢了解到了什么你发现承受不了了呢？是哪些底层概念的分歧，底层逻辑的不匹配我们解决不了或者没办法满足你的需求呢，又是否是这些原因导致了感情的变质？即便是在一起的过于唐突了，在好多东西还不了解还不确定的情况下你又为什么很确定的给出了承诺呢？如果你不那么确定，我们只是试探的往前走，是不是现在伤害就没有那么大。

    我想表达的是，一段关系的结束或破裂，其根本一定在于关系双方的需求无法匹配或者得到满足（需求不能满足一方面可能是底层逻辑没有达成共识，另一方面可能是感情的变质），因此我ex的“在一起的过于唐突了”的总结让我非常的不适。而我用对“对爱情的渴望让包法利夫人在不了解包法利先生的情况下急于嫁给他”解释的是包法利夫人婚前婚后对丈夫态度的转变，而且将关系推进到婚姻里面在很大程度上也比较难回头（在当时的背景下），而包法利夫人婚姻的不幸依然是她的需求没有被看到和满足。所以当爱情走过蜜月期，爱情才刚刚开始。

  • 与罗多尔夫
    这段关系我没什么可说的，因为它里面根本没有感情可言。从一开始，罗多尔夫就是在欺骗，就只是把包法利夫人当成猎物在去享受征服的快感。我不得不承认读到这部分内容的时候，我的内心是恐惧的。因为现实生活中也一定是有这样的人的，而我们能做的，就是提高自己的判断力，这需要自我意识的觉醒，需要有其他方面的追求（事业），需要足够的自信与冷静。可即使这样，我们也无法保证永远不被这样的人欺骗与控制，我们唯一能做的，就是相信自己的感受，有自我觉察的能力，不要听对方做了什么，而是看对方做了什么，把更多的注意力放在自己身上，遇到这样的人也不是我们的错，一旦有不舒服的地方，相信自己，重新审视一下这段关系和这个人。

  • 与莱昂
    这段关系是让我感触最多也最深的。首先是因为和与罗多尔夫不同，他们之间（至少是莱昂对包法利夫人）是真实存在过爱情的，想念却小心翼翼不敢逾越一步的状态我想我们都曾体会过。可是爱玛忽略了一点，也是我很久以来没有意识到的一点是爱情不是静止的，它也是个动态的过程。因此当莱昂再次出现在包法利夫人面前时，我不能说他不爱了，只是这份爱没有最初那么纯粹了，它里面增加了太多的杂质，比如肉欲、征服欲等等。所以在短暂的电光火石之后，爱玛依然会觉得是平凡且平庸毫无特别的，所以当爱玛要求莱昂事事汇报，给她写情书，时时向她表达心意会让莱昂觉得害怕甚至厌烦，他有杂质的爱里面是只够消遣而承载不了这么多东西的，所以当最后爱玛走投无路的时候，他选择的是逃避和远离（和大多数男人一样）。也许要承认的是有的人的爱真的就只有这么多，承载多了的时候就会承载不了然后选择丢掉逃跑。

脱离经济谈问题是没有意义的

这是最后一点我想说的，毕竟最后让包法利夫人走投无路的直接原因是负债。随然我对金钱没有强烈的追求欲，但是起码的财富自由是让我们去追求自我的保障。没有财富自由，我们就势必要依赖于别人，这时我们拿什么去谈自我，去谈自我价值，去谈自我追求。所以无论如何，财富自由是前提，这一点不用多说什么。

理解一个人需要漫长忘我的付出

发表于 2021-04-09 更新于 2021-04-12 分类于 生活体悟

之前在播客上听到的这句话：理解一个人需要漫长忘我的付出，而误解只要一瞬间。一下子就被它抓住了，它便成了这段时间我各大社交软件上的个人签名。

起因是我表哥家前两天爆发了世纪大战。我哥指责他的父母（就是我姑和我姑父）在人生的关键路口没有给他正确的指引，包括大学的选择，专业的选择，留学的选择，工作的选择，导致他现在本科毕业后在家待了好几年现在国外出不去，国内又找不到称心的工作。顺便也把多年来对我姑和姑父的金钱观带着怨气的贬低了一通。我姑和姑父自然生气又伤心，他们说我们一直尊重你的选择，考大学的时候是你说要离家越远越好才去的江西不然我们给你报个大工盘锦分校多好，选专业的时候也经过了你的同意，去美国半年我们也给你拿了钱，你想出国我们也让你在家学法语养着你，是你自己法语没有学出来出不了国，没有能力找不到好工作，我们能付出的都付出了，你怎么现在能把责任全都推给我们……听我姑给我讲这些的时候我觉得很遗憾，我没有办法做出任何判断，我没有办法站在谁的一边，甚至我觉得说你们两边在这件事情上都有责任都是不负责任的。我确实真实的认为双方的情绪和感受都很合理，我不能说我了解事情的全貌，但我可以看到在我哥这二十多年的成长过程中他的原生家庭给他带来的创伤，一直在扮演着听话的孩子的角色去满足父母的认可，一直被拿来与所谓“更优秀”的妹妹（就是我）作比较，一直无法认同父母的价值观又得顺从并且某种程度上也吸收了部分，没有被给予一个被鼓励表达自我的环境，而胆小谨慎的时候又会被嘲讽挖苦……我经常想换做是我我可能做的还不如他好。那站在我姑和姑父的角度，他们对我哥的付出无论从金钱上还是从感情上也都是毫无保留的，这点也毋庸置疑，只是他们习得的一些观念和习惯在他们也没有意识的情况下影响了我哥，在他性格和人格的塑造和形成上留下了一些印记。他们都没有做错什么，但却都在伤害与被伤害。作为局外人我没有办法去评论这整件事情，一是因为本身没有对错，二是我虽然说了这么多也只是看到了事情呈现在我面前的样子，没有处在事情的中心，任何论断都极其自负与不负责。

但我却很想跟他们分享，可不可以稍微试着理解那么一点对方。我试着跟我姑表达，我很理解你此时此刻的感受，并且我认为这是完全合理的，但同时我也理解我哥的感受，我觉得他的感受也完全合理，你们都没有做错什么，但是我们要承认每个人都有局限性，所以可不可以试着去理解一下他的过往，理解一下他为什么现在觉得承受了这么大的压力，理解一下为什么他的性格是你们口中所说的软弱不自信，能不能看看他经历了什么，哪些事情让他产生了哪些感受，他又为什么选择时时压抑自己无法面对自己。但让我也意外也不意外的事情是，好像从我嘴里说出的每一个字都是阿拉伯语他们完全听不懂没办法理解，他们不懂我为什么要理解他，不懂他怎么会有创伤，不懂这件事情怎么可能需要我们共同去面对去解决。我一边试着跟她分享我所看到的，一边意识到了我姑的思维固化也感受到了过程中我渐渐生出了一种无力感。我真的没有办法改变一个人，尤其是当她不愿意改变的时候（都不用改变，起码是愿意了解）。

我也尝试叫我哥出来喝个下午茶唠唠，但依然让我感到无力的是除了拒绝了我的邀请，他甚至还把我的邀请当成是一种官方的仪式，然后返回给我一套官方的拒绝说辞。虽然有点无奈，但我好像已经慢慢接受了每个人有每个人的选择这件事情。那如果别人不想听你分享观点，你当然不能把它们强加给对方，当他们选择不想的时候，我会尊重他们的选择，只不过作为家人我会有一点点的遗憾。

我想分享的，只不过是，生活里，尤其是原生家庭会发生很多创伤性的事件，而造成创伤和承受创伤的双方可能都是无意识的，但这种创伤的体验一旦有了，便会无意识的在无数个当下发挥作用。如果我们能察觉出这些创伤，察觉出这些创伤背后的模式，我们便可能找到方法从里面跳出来，放过别人，也放过自己，这个过程如果能做到试着去理解，便无关指责，说的鸡汤一点，算是用爱去治愈创伤？（有点过于鸡汤了，但确实是这样。）

理解也从来是比想象中更难做到的事情，因为这意味着你要放下自我，跳出自己的框架，去用对方的角度思考问题，这是一件让人不那么习惯的事情，甚至有点害怕的事情，也不是一件所有人都可以学会的事情。所以最后再回归到那句话：理解一个人需要漫长且忘我地付出，而误解只要一瞬间。

问

发表于 2021-04-06 更新于 2021-04-14 分类于 叨叨叨叨

今早在做冥想的时候，脑袋里又把最近发生的事情过了一遍，当然，主要还是一遍遍反复的追问。

  • “是先有感情的消失后有面对问题的逃避，还是先有面对问题的逃避再有情感的消失？”

  • “感情又是怎么消失的呢？是一开始就不够浓烈还是被矛盾消磨没的？是像大家说的那样是必然且必经的过程吗？”

  • “我所追寻的情感上的连接是否真的是有意义的呢？我一直探求的所谓的真理，所谓的世界运转的规则，是否真的有必要去了解呢？”

  • “当别人都去选择不了解，我又如何处理我的孤独感，我又如何去与人连接？”

  • “我不喜欢给人下定义，但我生命中出现的人我好像在心里又默默给他们打上了标签”

……

面对这些天问我显然是还没有找到答案的。但追问这个行为本身我想有他的意义，追问好像是我与自己连接的方式，追问让我意识到自己的存在，追问是让我弄清楚我是谁的第一步。自己问自己的过程也许让我察觉到我此时此刻在乎的东西是什么（可能他其实不重要），也让我更能看到我自己和外界，让我感受到我是一个可以思考的生命体，我对自己和世界都还充满好奇，这难道不算一种生命力吗？对于这种追问，很多人会认为“想太多没必要”，或者“这样会让你活的很累”，甚至“你是不是有点偏激了”，但我认为，也是我听到的一期播客分享的一个点：只有追问，才能让我们离我们一直以来所探寻的答案更靠近一点。追问让我寻找，追问让我探索，追问让我理解。我也相信只有更多的了解事物运行的规则，了解自己运行的规则，才能看到更多的可能性，然后理直气壮地做出自己的选择。

如果在上一段关系结束的时候我没有问“我们之间到底发生了什么，怎么就变得没有余地了”，我不会发现我们两个人对感情的需求是如此的不同，不会明白我永远没有办法改变一个人，当然也不会看清对方的自负以及自己的不成熟。我不会看到自己一直也因为没有被看到而感到委屈和愤怒，我不会停下来好好再看看自己。如果在患有抑郁以后我没有问“我的抑郁从哪里而来，到底什么东西出了问题”，我不会理解我原生家庭的模式，不会理解原生家庭对我产生了哪些影响，我又习得了哪些思维和模式，我不会有“自我”的意识。如果我不问自己想要什么我就永远不知道自己想要什么，如果我不问自己的情绪从何而来我就永远看不见我情绪背后的需求，那这些需求自然也就不太可能被满足。

归根结底，追问让我看到自己。

胡言乱语地第一次尝试

发表于 2021-04-05 分类于 叨叨叨叨

还在学习博客的搭建过程，包括怎么发布文章，怎么设计界面，怎么放资源等等。搭建这个博客的初衷，是想通过写作的方式去整理自己的点滴，包括过往的经历，生活的体悟，奇奇怪怪的一些小想法，以及时常存在于我身体里的那么真实的情绪和感受。他们需要被看到，被我看到。我想好好看看这个小女孩，重新的认识她，陪伴她，理解她，对她的一切都充满好奇，我不会评判她的任何一个决定，不会否定她的任何一种感受，不会对她抱有任何期待，我只想用开放的心态去探索有关于她的一切，探索她的身体，探索她身体里蕴含的能量，探索只关于她的生命力。

写博客的过程是练习的过程，练习更好的表达自己，练习时时保持对自我的觉知，时时内省，看到那些隐蔽的思维模式和行为模式，虽然很多时候他们很好的保护了我，但我想要先认识他们，然后谢谢他们，最后告诉他们那个受过伤的小孩已经长大了，有能力去选择更健康的方式去保护自己了。

在这里我会试着勇敢的袒露自我，真诚的面对自己，也真诚的面对在我生命里出现的每一个值得的你们。
